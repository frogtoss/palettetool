Palettetool is a binary program that does conversions to/from JSON
palette format from known binary formats in a structured, rigorous way.

The tools directory contain experimental and heuristic exporters from
various fromats.  For example, it contains a .css source you can paste
into your Chrome developer window that generates a palette json from
the current CSS colors.  This can be used as a starting point to
capture a palette.

These sorts of experiments and hacks are useful for recovering colors,
but are fallible.  The binary 'palettetool' program is designed to be
reliable, and these scripts are designed to be experimental.
